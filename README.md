# 📊 Sales Forecasting System

## 🌟 Overview
The **Sales Forecasting System** predicts future sales using historical data.  
It uses **Machine Learning and Time Series models** to analyze patterns and generate accurate predictions.  

The system also provides **graphs and dashboards** to make results easy to understand.

---

## 🎯 Objectives
- Predict future sales  
- Improve inventory and resource planning  
- Support better business decisions  
- Identify trends and seasonal patterns  
- Reduce risks like overstocking and shortages  

---

## 🔄 System Workflow
📁 Data Collection
↓
🧹 Data Preprocessing
↓
📊 Data Analysis (EDA)
↓
⚙️ Feature Engineering
↓
🤖 Model Training (ARIMA, Linear Regression)
↓
📉 Model Evaluation (MAE)
↓
🔮 Sales Forecasting
↓
📈 Visualization & Dashboard

---

## ✨ Key Features
- Easy CSV file upload  
- Sales trend visualization  
- Future sales prediction  
- Model comparison (ARIMA & Linear Regression)  
- Interactive dashboard using Streamlit
